# Linux WSL Environment

## Objective

Deploy and utilize a Linux environment through Windows Subsystem for Linux (WSL) to perform software preparation, networking activities, system administration, and infrastructure engineering tasks.

---

## Environment

- Microsoft Windows
- Windows Subsystem for Linux (WSL)
- Linux Terminal
- Bash Shell
- Virtual Laboratory Environment

---

## Activities Performed

- Installed and configured Windows Subsystem for Linux (WSL)
- Configured the Linux environment for engineering activities
- Updated and upgraded Linux packages
- Downloaded software packages and source code from official repositories
- Managed files and directories using Linux commands
- Transferred files between Windows and Linux environments
- Prepared software packages for deployment
- Transferred prepared files into an isolated laboratory environment
- Verified file integrity after transfer
- Performed command-line administration and troubleshooting
- Documented installation and deployment procedures
