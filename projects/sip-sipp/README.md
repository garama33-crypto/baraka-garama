# SIPp Protocol Testing

## Objective

Perform SIP protocol testing using SIPp to simulate VoIP traffic and validate SIP signaling behavior in a controlled laboratory environment.

---

## Environment

- Linux
- SIPp
- Terminal
- Virtual Lab Environment

---

## Activities Performed

- Installed and configured SIPp
- Generated simulated SIP calls
- Executed SIP scenarios
- Produced SIP messages for testing
- Validated signaling procedures
- Analyzed protocol responses

---

## Technologies Used

- SIP
- SIPp
- VoIP
- Linux
- Terminal

---

## Skills Demonstrated

- SIP protocol testing
- VoIP traffic simulation
- Network troubleshooting
- Test execution
- Log analysis
- Telecommunications testing

---

## Test Scenarios

- SIP INVITE
- SIP REGISTER
- SIP ACK
- SIP BYE
- Call establishment
- Call termination

---

## Lessons Learned

Using SIPp enables realistic simulation of VoIP traffic and provides valuable experience in SIP signaling, protocol analysis, and telecommunications testing within a safe laboratory environment.
