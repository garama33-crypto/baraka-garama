# Telecom Network Intelligence Agent

## Overview

An AI-powered Telecom Intelligence Agent built with n8n and OpenAI.

The agent automatically:

- Monitors telecom industry news
- Tracks IMS and SIP developments
- Identifies security advisories
- Summarizes findings
- Delivers intelligence reports by email

## Architecture

```text
Trigger
   ↓
AI Agent
   ↓
OpenAI Chat Model
   ↓
Gmail
```

## Focus Areas

- IMS
- SIP
- SBC
- PSTN
- VoIP
- Telecom Security
- AWS Networking
- Azure Networking

Prompt engineering and source selection have a greater impact on agent quality than the AI model alone.
