# Asterisk PBX Deployment

## Objective

Deploy, configure, and administer an Asterisk PBX environment from source code to gain practical experience in VoIP communications, SIP signaling, and Linux system administration.

---

## Environment

- Linux
- Terminal
- Source Code Compilation
- Virtual Laboratory Environment

---

## Activities Performed

- Downloaded Asterisk source code
- Installed and configured required dependencies
- Compiled Asterisk from source
- Installed and configured PBX services
- Configured SIP/PJSIP endpoints
- Created and managed VoIP users
- Configured dialplans
- Created and managed extensions
- Defined call routing logic
- Tested extension-to-extension communication
- Validated call establishment and termination
- Performed troubleshooting and log analysis
- Managed Asterisk services through the Linux terminal

---

## Dialplan Configuration

### Activities

- Created and modified Asterisk dialplans
- Configured extensions for internal calling
- Defined call routing logic
- Implemented basic call handling rules
- Tested extension-to-extension communication
- Validated dialplan execution through simulated calls
